Modifier les variables dans Vagrantfile :

`PHP_VERSION = "7" # 5.4, 5.5, 5.6 or 7`

`WWW_DIRECTORY = "D:/www"`

Ajouter au fichier host de windows (C:\Windows\System32\drivers\etc\hosts) les lignes

`127.0.0.1	local.dev`

`127.0.0.1	www.local.dev`

`127.0.0.1	db.local.dev`
